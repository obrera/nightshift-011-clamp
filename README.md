# nightshift-011-clamp

Clamp Craft — a tiny TypeScript + React utility for generating responsive `clamp()` typography formulas.

## What it does

- Calculates fluid font-size CSS with `clamp(min, preferred, max)`
- Lets you tweak min/max viewport and font-size ranges
- Includes quick presets (Body, Heading, Display)
- Interactive preview slider to inspect computed font size at different widths
- One-click copy of generated CSS

## Run locally

```bash
bun install
bun dev
```

Build production bundle:

```bash
bun run build
```
